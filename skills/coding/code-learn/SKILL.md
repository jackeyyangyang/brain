# Code Learn

代码修改后自动生成变更说明 + 知识点提炼，存入 brain/learnings/ 学习库。

## 触发

- `/code-learn`
- "消化这段修改"
- "解释这次改动"
- "学习这段代码"

## 流程

### 1. 获取 diff

```bash
git diff HEAD~1 --no-color
```

### 2. 生成三段式总结

对每一处修改，输出：

```
## [文件名] - [简短描述]

### 改了啥
- 具体改了什么

### 为啥这么改
- 设计意图
- 为什么这样做而非其他方案

### 学到了啥
- 涉及的语法/库/模式
- 设计思路
- 可复用的模式
```

### 3. 写入学习库

追加到 `brain/learnings/YYYY-MM-DD.md`，格式：

```
# 2026-06-19

## [文件名] - [简短描述]
**改了啥**：...
**为啥改**：...
**知识点**：...
```

## 原则

- 简洁，不废话
- 知识点要有深度，不只是描述代码
- 设计意图是核心，帮助理解"为什么"
